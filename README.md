### Arduino Password Counter

Sistema contador de senhas desenvolvido utilizando Arduino UNO, Display OLED, botões e buzzer.

O projeto permite incrementar ou decrementar a numeração apresentada no display por meio dos botões físicos, oferecendo também feedback sonoro para cada interação realizada pelo usuário.

### Tecnologias Utilizadas:
Arduino UNO


C++

Display OLED (SSD1306)

Buzzer

Botões

### Funcionalidades:

Incremento e decremento de valores;

Exibição em Display OLED;

Feedback sonoro;

Controle do limite mínimo e máximo da numeração apresentada.

Desenvolvido para fins acadêmicos e de aprendizado.
